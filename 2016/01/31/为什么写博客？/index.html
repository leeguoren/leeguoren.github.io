前言

为什么写博客？ 这是一个没有标准答案的问题，每个人心中或许都有自己的答案。对于我来说，我写博客是为了记录学习和生活、更深入地思考等等。

写博客有哪些好处

  1. 记录学习和生活
  2. 促进思考
  3. 保持持之以恒地做一件事情的习惯
  4. 推销自己
  5. 交到许多志同道合的朋友

自己的博客历史

记得大一暑期刚来实验室的时候，学长让我们做的第一件事情，不是敲题而是建个自己的博客，用来记录 ACM 的解题思路。这便是我第一次接触博客，当时不懂为何还要记录解题思路。学长说的就照做了，于是便有了我的第一个 博客 （里面记录400多道的题解，还有几十篇的软件的安装与配置）。 写博客久了后，慢慢体会到了它的种种好处，但是由于大多数博客网站的个性化低，于是便想建立一个属于自己的博客，但是买服务器和域名又麻烦，最后选择用 GitHub 来建立自己的博客（免费提供 300M 的空间，省去了买服务器和域名的麻烦），在物色各种主题后，最终选择了 jacman 这个主题（简洁大方），这便是本博客。

本博客的定位

主要用来记录自己的各种软件的配置，还有学习和生活的记录。

最后

希望通过本博客交到许多志同道合的小伙伴们！欢迎你们的来访！
